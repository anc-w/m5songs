**M5Songs** is a collection of songs playable on the **M5StickCPlus2** using Arduino IDE. Each song is converted from MIDI files to Arduino-compatible note sequences ‚Äî no copying from others, 100% original conversions.  

---

## Features

- Play songs on the **M5StickCPlus2** using a buzzer.  
- Lightweight and optimized for ESP32-based M5 devices.  

---

## Requirements

- **Hardware:**  
  - M5StickCPlus2  

- **Software:**  
  - Arduino IDE (latest version recommended)  
  - **ESP32 Dev Module** installed via Arduino Board Manager  
  - **M5Unified** library  

---

## Installation

1. Install the [Arduino IDE](https://www.arduino.cc/en/software).  
2. Install the **ESP32 Dev Module** via **Boards Manager**.  
3. Install the **M5Unified** library.  
4. Copy the `.ino` file of the song you want to play.  
5. Open the `.ino` file in Arduino IDE.  
6. Select **ESP32 Dev Module** as the target board.  
7. Upload to your M5StickCPlus2.  

---

## Usage

1. Power on your M5StickCPlus2.  
2. Press the **M5 Button** to play the song.
3. Press the **Next Button** to pause/resume the song.
